Any Questions?

We encourage that you direct any questions or feedback to the SNPsnap team. Thank you for helping improving SNPsnap

Input

Will I be able to use rs-numbers as input SNP identifers?
No. SNPsnap only accepts chromosomal coordinates (from GRCh37/hg19) as SNP identifiers.

Output

What gene annotations does SNPsnap use for column locus genes likeID_genes_in_matched_locus?
SNPsnap uses the complete set of genes from Ensembl which covers about 48,000 genes.
Why is there a replicated input SNPs in matched_snps.tab even though I enabled Exclude input SNPs in matched SNPs?
If no matching SNPs are found, SNPsnap will replicate the input SNP to form a full row in matched_snps.tab.

Getting results files

What are the run times for SNPsnap?
SNPsnap matching takes ~3 seconds per input SNP (matching 500 input SNPs will take ~25 minutes to match). If any annotation option (Annotate matched SNPs or Annotate input SNPs) option is enabled SNPsnap uses an additional ~2 minutes on annotation for the complete set of SNPs.
For how long are my files saved on the web server?
The SNPsnap web server stores the result files for 24 hours. After this your result files will be permanently deleted.
