SNPsnap main usage

Match input SNPs
SNPsnap can be used to generate matched sets of SNPs that can be used to calibrate background expectations in SNP-based enrichment analysis.

Annotate input SNPs using 1000 Genome data
SNPsnap allows users to quickly retrieve 1000 Genome population specific information about loci bounderies, nearest genes and genes in loci, minor allele frequency and much more.

Tips & Tricks

Tip #1: fast input annotation
SNPsnap main purpose is to match input SNPs. However, many users are often only interested in annotating their input SNPs using 1000 Genome population specific data. If you are only interested in annotatating your input SNPs, setting Number of matched SNPs to a low value (e.g. 1) will highly speed up the runtime for your job. Of course you should also make sure to enable Annotate input SNPs.
